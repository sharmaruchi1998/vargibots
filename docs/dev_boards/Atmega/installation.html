Installation of Software

Requirements

The installation of the software are tested on Windows 8 OR 10 (with 64-Bit), Ubuntu 16.04 OR 18.04 (with 64-Bit). You may choose to use either OS and follow the instructions respectively.

Installation Instructions

This section contains instructions to install following software/libraries:

  1. Visual Studio Code
  2. AVR Building Tools
  3. AVRDUDESS

1. Visual Studio Code:

Visual Studio Code is a cross-platform free and open-source code editor made by Microsoft. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring and embedded Git. Users can change the theme, keyboard shortcuts, preferences and install extensions that add additional functionality.

Installation Steps:

For Ubuntu

  • Download VS Code for Linux for 64-bit Linux from this page. The software with latest version named code_x.y.z-abcdefgh_amd64.deb (here, letters are digits of VS Code major and build version) will be downloaded.

  • Open Terminal and navigate to the directory where this file is downloaded. Run the following command on the Terminal:

    sudo dpkg --install code_1.55.1-1617808414_amd64.deb

    Note: The .deb file name may be different at your end as this software is timely updated.

For Windows

  • Download VS Code for Windows from this page.

  • Navigate to the directory where the above .exe file is downloaded.

  • Run the .exe file by double clicking on it.

VS Code is now installed on your machine.

Open VS Code by searching for it after clicking the Show Applications icon at the bottom-left corner.



Refer this document to learn about the User Interface of this software and their use.

2. AVR Building Tools

For Ubuntu
  • Install AVRDUDE and AVR-GCC: 
    sudo apt-get install avrdude gcc-avr avr-libc
For Windows

  • Download WinAVR zip file and extract it.

  • Run .exe file to install AVR-GCC Toolchain and AVRDUDE.

3. AVRDUDESS ‚Äì A GUI for AVRDUDE

AVRDUDESS is a GUI for AVRDUDE, a tool for programming Atmel microcontrollers.



Installation Steps:

For Ubuntu

  • Install Mono:

    sudo apt-get install libmono-system-windows-forms4.0-cil

    On older versions of Ubuntu you might need to use libmono-winforms2.0-cil instead.

  • Download AVRDUDESS zip file and extract it.

  • Run AVRDUDESS with mono, you might have to run as root (sudo) so that AVRDUDE can access ports if you haven‚Äôt changed any permissions or rules.d stuff:

    mono avrdudess.exe
For Windows

  • Download AVRDUDESS .exe file.

  • Double click on .exe file to open a GUI.
